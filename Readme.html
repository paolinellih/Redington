Backend: CalculatorAPI

A C# .NET 8 API that provides probability calculation services and other mathematical endpoints.

Prerequisites

  • .NET 8 SDK
  • Visual Studio 2022 or any compatible IDE

Running the API


git clone <https://github.com/paolinellih/Redington>
cd CalculatorAPI

dotnet restore
dotnet build
dotnet run

The API will start on the following URLs:

  • HTTP: http://localhost:5000
  • HTTPS: https://localhost:44300

Frontend: React.js with VITE

A React.js application built using the VITE toolchain.

Prerequisites

  • Node.js
  • npm or yarn

Running the Frontend


git clone <repository-url>
cd frontend

npm install
npm run dev

Open your browser and navigate to the URL provided by the VITE server, typically:

http://localhost:5173

Connecting Backend and Frontend

Update the backend API base URL in the frontend configuration file, for example:


export const API_BASE_URL = 'https://localhost:44300';
